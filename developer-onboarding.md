# Developer Onboarding

## Prerequisites

- Python 3.11+
- uv package manager

## Setup

1. Clone the repository
2. Run `make bootstrap`
3. Make your changes
4. Run `make test` to verify
5. Run `make lint` and `make format`

## Project Structure

- `src/harness_precommit/`: Main package
- `tests/`: Test suite
- `harness-schema/`: Schema submodule
